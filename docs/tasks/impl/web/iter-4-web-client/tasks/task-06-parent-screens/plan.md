# Task 06: Parent web screens — plan

**Iteration:** iter-4-web-client

## Goal

Build guardian-facing pages: overview of linked children, per-child activity, and per-child progress aligned with backend guardian routes.

## Scope

- **ChildrenPage:** Grid or list of children with navigation into detail routes.
- **ChildActivityPage:** Summary stats and recent conversations (or equivalent activity feed).
- **ChildProgressPage:** Snapshot/progress table for the selected child.

## Dependencies

Guardian API (Task 03); parent routing (Task 04).

## Out of scope

Student-only screens (Task 05); new backend endpoints unless strictly required.
