# Task 05: Student web screens — plan

**Iteration:** iter-4-web-client

## Goal

Deliver the main student-facing web UI: browse conversations, chat with real-time feel, and review learning progress via knowledge snapshots.

## Scope

- **ConversationsPage:** List conversations (e.g. with dates) from the API.
- **ChatPage:** Load message history, send new messages, UX polish (e.g. typing indicator) where implemented.
- **ProgressPage:** Table of knowledge snapshots, reusing shared components (e.g. `LevelBadge`) for levels.

## Dependencies

Authenticated student routes (Task 04); list endpoints (Task 03).

## Out of scope

Parent/guardian screens (Task 06).
