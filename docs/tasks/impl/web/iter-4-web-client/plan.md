# Итерация 4: Веб-клиент ученика и родителя — план

**Область:** web
**Tasklist:** [docs/tasks/tasklist-web.md](../../../tasklist-web.md)
**Дорожная карта:** [docs/plan.md](../../../../plan.md)

## Цель

SPA веб-клиент (`web/`) для ученика и родителя, подключённый к единому backend REST API. Ученик видит историю занятий, ведёт диалог с репетитором, отслеживает прогресс. Родитель видит активность и прогресс ребёнка.

## Ключевые решения

- **Стек:** React 19 + Vite + TypeScript + Tailwind CSS + TanStack Query + React Router ([ADR-004](../../../../adr/adr-004-web-stack.md))
- **Аутентификация:** JWT + одноразовые коды (MVP); заготовка под OAuth
- **Двойная идентификация:** backend поддерживает и заголовки (бот) и Bearer JWT (веб)

## Задачи

1. Стек и каркас веб-клиента (ADR, scaffold, Makefile)
2. Аутентификация backend (JWT, коды, эндпоинты auth, команда /web)
3. API для веб-сценариев (список диалогов, прогресс, родительские эндпоинты)
4. Web: аутентификация и роутинг
5. Web: экраны ученика
6. Web: экраны родителя
7. Интеграция, тесты, документация
